# 💰 Expense Tracker

## 📌 Description

The Expense Tracker is a simple application that helps users record and manage their daily expenses. It allows users to keep track of spending, categorize expenses, and view their total expenditure.

---

## 🚀 Features

* ➕ Add new expenses
* 📋 View all expenses
* 🗑️ Delete expenses
* 🏷️ Category-based tracking (Food, Travel, Bills, Shopping)
* 💵 Calculate total spending

---

## 🛠️ Technologies Used

* Java
* Object-Oriented Programming (OOP)
* File Handling / Console-based input-output

---

## ▶️ How to Run

1. Clone the repository
2. Open the project in any Java IDE (Eclipse / IntelliJ / VS Code)
3. Compile the program
4. Run the main class

---
